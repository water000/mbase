功能介绍

守望者APP是由江苏省安全厅研发的一款面向广大公民开发的一个提供信息来源的应用，是以广大人民群众为使用对象，可以查看国家安全机构所公布的任务信息，根据自己个人能力并且在能力范围之内协助国家安全部门提供有价值的情报和线索，对任务发布方审核通过并采纳以后会有一定数额现金奖励。此外该平台还提供资讯类信息和服务，内容涵盖法律法规、安全教育、新闻动态和视频点播等多个资讯类别，其内容更加专业、全面和精致。

任务列表

查看国家安全部门发布的任务和奖励信息

我的任务

参与并响应国家安全部门发布的任务，并积极提供有价值的情报和线索

安全资讯

提供资讯类信息和服务，内容涵盖法律法规、安全教育、新闻动态和视频点播等多个资讯类别，其内容更加专业、全面和精致。

在线举报

欢迎广大公民提供有关间谍嫌疑的线索，为国家安全提供必要的信息和帮助，共同筑牢国家安全的人民防线。
